Проект ПМ02 Информационная система учета заказов такси включает в себя:
=======================================================================
-----------------------------------------------------------------------
- database.py - файл базы данных
- menu.py - главное меню
- main.py - главный файл
- user.py - работа с пользователем
- order.py - работа с заказом
- driver.py - работа с водитлями
- car.py - работа с автомобилями
- confing.py - имя базы данных
------------------------------------------------------------------------
Возможности Информационной системы:
========================================================================
Регистрация пользователей в системе
Создание и обработка заказов
Работа с пользователя.
------------------------------------------------------------------------
Преимущества Информационной системы:
========================================================================
Удобный интерфейс
Все данные храняться в одном файле
Разделение пользователей по ролям
Автоматизация учета заказов.
------------------------------------------------------------------------
Результат работы:
========================================================================
Создание полноценной информационной системы
Реализованы роли для каждого пользователя
Обеспеченно управление заказами
Система готова к дополнительным расширениям.
------------------------------------------------------------------------
